# FocusFlow

Esta es una página de aterrizaje sencilla para **FocusFlow**, una aplicación de productividad. El proyecto fue creado para practicar maquetación web y el uso de SASS.

##  Tecnologías usadas
- **HTML5**: Para la estructura de la página.
- **SASS**: Para los estilos y el diseño.

##  Cómo ver el proyecto
1. Clona o descarga este repositorio.
2. Abre el archivo `index.html` en tu navegador favorito.

##  Estructura de carpetas
- `index.html`: Archivo principal con el contenido.
- `scss/`: Contiene los archivos de estilo originales.
- `scss/estilos.css`: El archivo de estilos final generado por SASS.
